===== README.md =====
# SuperOwl Voice AI — Prototype

Internship project. A standalone voice AI prototype for SuperOwl clients, built on top of VAPI. Tested with **Troy Blocks** (educational wooden toy company, Bengaluru).

---

## What It Does

When a customer requests a callback on the chat widget, the system:

1. Collects their name, phone, and chat summary
2. Calls the customer via an AI voice agent (Ananya)
3. Ananya continues the conversation, answers product/pricing questions using a RAG knowledge base
4. When the customer shows buying intent, Ananya silently notifies the backend
5. Backend calls the business owner via a second AI agent (Owner PA)
6. Owner PA briefs the owner and asks if they're available
7. If yes — the customer call is transferred to the owner's phone in real time
8. If no — Ananya politely wraps up the call

---

Two VAPI assistants, both with prompts injected at call time (nothing hardcoded in dashboard):
- **Ananya** — customer-facing, warm and conversational, uses RAG for product knowledge
- **Owner PA** — owner-facing, concise and professional, just briefs and gets a yes/no

Transfer is done via a direct POST to VAPI's `monitor.controlUrl` — not via a tool call, which was unreliable.

from kb.embeddings import search_chunks
from dotenv import load_dotenv
load_dotenv()

client=Groq(api_key=os.getenv("GROQ_API_KEY"))
system_prompt="""You are a helpful AI assistant for Troy Blocks, an educational wooden toy company based in Bengaluru.
Answer questions based ONLY on the provided context from the knowledge base.
If the answer is not in the context, say "I don't have that information, please contact us at 9731737460."
Be professional, friendly, and concise.
Do not make up information.
If user is a preschool owner, emphasize structured curriculum and workshop model.
If user is a parent, highlight creativity, birthday parties, and learning through play.do not be presumptous,and dont exceed more than 3 lines.
If the conversation leans towards the purpose of knowing more,or booking,only then callback."""

def chat(question:str,hist:list[dict],namespace:str="superowl-proto")->str:
    chunks=search_chunks(query=question,namespace=namespace,top_k=3)
    context="\n\n".join(
    [f"[Source :{c['page_title']}]\n{c['text']}"
    for c  in chunks]
    )
    messages=[
        {
            "role":"system",
            "content":system_prompt
        },
        *hist,
       {
            "role": "user",
            "content": f"""Context from knowledge base:{context} Customer question: {question}"""
        }
    ]
    response=client.chat.completions.create(messages=messages,model="llama-3.1-8b-instant",temperature=0.2)
    return response.choices[0].message.content

Phone numbers must be spoken **digit by digit**.

Example:

"nine eight seven, six five four, three two one zero"

Never add country code.

---

# Turn Discipline

• Ask **one question at a time**
• **Wait 2 seconds for response**
• If no response after 2 seconds: **repeat the question once, softly** (e.g., "Sorry, did you have a chance to think about that?")
• If **still no response** after second ask: **assume "no" or offer callback** ("No worries, we can call you back!")
• If caller says "one moment", **wait silently for 5 seconds**, then gently: "Still here whenever you're ready."
• Never ask same question 3+ times in a row

---

# Identity Rule

If asked who you are:

"I'm a virtual assistant helping coordinate Troy activity bookings."

If asked if you are AI:

"Yes, I'm a virtual assistant helping with booking coordination."

Keep explanations short.

---

# Silence Handling

**If customer goes silent > 3 seconds:**

Say: "I'm still here. Are you there?" or "Hello?"

**If silence continues > 2 more seconds:**

Say: "It seems like we lost the connection. No worries — we'll call you back!"

Then call end_call_tool().

---

# Closing

When ending the conversation:

"Thank you for your interest in Troy activities. Have a wonderful day.""",

async def make_vapi_call(
    assistant_id: str,
    phone: str,
    name: str,
    system_prompt: str,
    first_message: str,
    transfer_number: str | None = None
):
    """
    Fire an outbound VAPI call.
    Injects system_prompt + first_message via assistantOverrides.
    Optionally injects transfer destination for transfer_call_tool.
    """
    url = "https://api.vapi.ai/call"
    headers = {
        "Authorization": f"Bearer {os.getenv('VAPI_API_KEY')}",
        "Content-Type": "application/json"
    }

    assistant_overrides = {
        "firstMessage": first_message,
        "model": {
            "provider":"openai",
             "model": "gpt-4o-mini",
            "messages": [
                {
                    "role": "system",
                    "content": system_prompt
                }
            ]
        }
    }

    # Inject transfer destination if provided
    if transfer_number:
       assistant_overrides["tools:append"] = [
        {
            "type": "transferCall",
            "function":{
                "name":"transfer_call_tool",
                "description":"Transfer the customer call to the business owner. Only call this when explicitly instructed after owner confirms availability."
            },
            "destinations": [
                {
                    "type": "sip",
                    "sipUri": transfer_number,
                    "description": "Transfer to business owner"
                }
            ]
        }
    ]

    payload = {
        "assistantId": assistant_id,
        "phoneNumberId": os.getenv("VAPI_PHONE_NUMBER_ID"),
        "customer": {
            "number": phone,
            "name": name
        },
        "assistantOverrides": assistant_overrides
    }

    async with httpx.AsyncClient(timeout=30.0) as client:
        resp = await client.post(url, headers=headers, json=payload)
    result = resp.json()
    print(f"VAPI call response ({resp.status_code}): {result}")
    return result

async def command_customer_call(customer_call_id: str, decision: str):
    tenant = get_tenant()

    if decision == "yes":
        session = call_sessions.get(customer_call_id, {})
        control_url = session.get("control_url")

        if control_url:
            # Tell Ananya to say the handoff line
            await send_background_message(
                customer_call_id,
                "The owner is available. Say exactly: 'Great news! The owner is ready to speak with you. Connecting you now, please hold.' then stop speaking."
            )

            # Small delay to let Ananya finish speaking
            import asyncio
            await asyncio.sleep(4)

            # Execute the transfer directly from backend via controlUrl
            headers = {
                "Authorization": f"Bearer {os.getenv('VAPI_API_KEY')}",
                "Content-Type": "application/json"
            }
            transfer_payload = {
                "type": "transfer",
                "destination": {
                    "type": "sip",
                    "sipUri": tenant["transfer_sip_uri"]
                }
            }
            async with httpx.AsyncClient(timeout=30.0) as client:
                resp = await client.post(control_url, headers=headers, json=transfer_payload)
            print(f"Transfer via controlUrl: {resp.status_code} {resp.text}")

        else:
            print(f"No control URL for call {customer_call_id}, cannot transfer")

    elif decision == "no":
        await send_background_message(
            customer_call_id,
            f"The owner is currently unavailable. "
            f"Say 'I'm sorry, our team at {tenant['business_name']} is a little occupied "
            f"at the moment. They will call you back shortly. Thank you so much!' then call end_call_tool."
        )

    elif decision == "voicemail":
        await send_background_message(
            customer_call_id,
            f"The owner did not pick up. "
            f"Say 'Our team seems to be unavailable right now but will call you back soon. "
            f"Thank you for your interest in {tenant['business_name']}!' then call end_call_tool."
        )

class CallbackRequest(BaseModel):
    name: str
    phone: str
    conversation_summary: str


@app.post("/trigger-call")
async def trigger_call(req: CallbackRequest):
    """
    Called by Streamlit frontend when customer requests callback.
    Fires Ananya call with full prompt + transfer destination injected.
    """
    tenant = get_tenant()

    if not tenant.get("transfer_sip_uri"):
        raise ValueError(
            "Transfer SIP destination is not configured. Set TRANSFER_SIP_URI or both OWNER_PHONE and TRANSFER_SIP_DOMAIN in .env"
        )

    # Full system prompt = base voice prompt + chat summary appended
    full_system_prompt = (
        tenant["voice_system_prompt"]
        + f"\n\n---\nChat Context (prior conversation with customer):\n{req.conversation_summary}"
    )

    # First message with customer name substituted
    first_message = tenant["voice_welcome_message"].replace(
        "{customer_name}", req.name
    )

    assistant_id = os.getenv("VAPI_ASSISTANT_ID")
    if not assistant_id:
        raise ValueError("VAPI_ASSISTANT_ID environment variable is not set")

    result = await make_vapi_call(
        assistant_id=assistant_id,
        phone=req.phone,
        name=req.name,
        system_prompt=full_system_prompt,
        first_message=first_message,
        transfer_number=tenant["transfer_sip_uri"],
    )

    call_id = result.get("id")
    if call_id:
        call_sessions[call_id] = {
            "customer_name": req.name,
            "customer_phone": req.phone,
            "conversation_summary": req.conversation_summary,
            "owner_call_triggered": False,
            "owner_call_id": None,
            "control_url": result.get("monitor", {}).get("controlUrl"),
        }
        print(f"Customer call triggered: {call_id}")

    return {"status": "call initiated", "call_id": call_id}

    pa_system_prompt = f"""[Identity]
You are a professional, efficient AI assistant representing SuperOwl.

[Style]
- Concise, clear, and professional.

[Customer Context]
Customer Name: {customer_name}
What they want: {call_summary}
Quality indicator: {lead_reason}

[Task]

            }
        ]
    }


@app.post("/owner-decision")
async def owner_decision(request: Request):
    """
    Called by Owner PA's report_decision tool.
    Receives owner's yes/no/voicemail decision.
    Commands Ananya via background message injection.
    """
    data = await request.json()
    print(f"Owner decision received: {data}")

    tool_call_id = data.get("message", {}).get("toolCallList", [{}])[0].get("id")
    arguments = data.get("message", {}).get("toolCallList", [{}])[0].get("function", {}).get("arguments", {})
    decision = arguments.get("decision", "").lower()
    owner_call_id = data.get("message", {}).get("call", {}).get("id")

    print(f"Decision: {decision}, owner_call_id: {owner_call_id}")

    customer_call_id = owner_to_customer.get(owner_call_id)

    if customer_call_id:
        print(f"Commanding customer call: {customer_call_id} with decision: {decision}")
        await command_customer_call(customer_call_id, decision)
    else:
        print(f"No customer call found for owner call: {owner_call_id}")

    messages = {
        "yes": "Transfer initiated. Customer is being connected to the owner now.",
        "no": "Noted. Customer has been informed the team will call back shortly.",
    }

    return {
        "results": [
            {
                "toolCallId": tool_call_id,
                "result": messages.get(decision, "Decision recorded.")
            }
        ]
    }

async def vapi_webhook(request: Request):
    """
    Receives all VAPI call events.
    Used for logging and call state tracking only.
    Qualification detection removed — Ananya decides via notify_owner tool.
    """
    data = await request.json()
    event_type = data.get("message", {}).get("type")
    call_id = data.get("message", {}).get("call", {}).get("id")

    print(f"VAPI webhook: {event_type} | call_id: {call_id}")

    if event_type == "call-ended":
        if call_id in call_sessions:
            print(f"Call ended. Session: {call_sessions[call_id]}")
        elif call_id in owner_to_customer.values():
            print(f"Owner PA call ended.")
    #this is for the summary
    if event_type == "end-of-call-report":
        msg = data.get("message", {})
        customer_name = msg.get("call", {}).get("customer", {}).get("name", "Unknown")
        customer_phone = msg.get("call", {}).get("customer", {}).get("number", "Unknown")
        duration_s = int(msg.get("durationSeconds", 0))
        duration_str = f"{duration_s // 60}m {duration_s % 60}s"
        ended_reason = msg.get("endedReason", "unknown")
        transcript = msg.get("transcript", "No transcript available.")
        started_at = msg.get("startedAt", "")[:16].replace("T", " ")
        assistant_name = msg.get("assistant", {}).get("name", "")

        print(f"END OF CALL REPORT: {assistant_name} | {customer_name} | {duration_str} | {ended_reason}")
        if assistant_name == "ananya":
            slack_url = os.getenv("SLACK_WEBHOOK_URL")
            if slack_url:
                slack_msg = {
                    "text": (
                        f"🤖 Ananya Call Report\n"
                        f"Customer: {customer_name} ({customer_phone})\n"
                        f"Duration: {duration_str}\n"
                        f"Reason: {ended_reason}\n"
                        f"Time: {started_at}\n\n"
                        f"Transcript:\n```\n{transcript}\n```"
                    )
                }
                def send_slack():
                    import httpx
                    r = httpx.post(slack_url, json=slack_msg, timeout=15.0,verify=False)
                    return r.status_code, r.text

                try:
                    loop = asyncio.get_event_loop()
                    status, text = await loop.run_in_executor(None, send_slack)
                    print(f"Slack response: {status} {text}")
                except Exception as e:
                    print(f"Slack notification failed (non-fatal): {type(e).__name__}: {repr(e)}")
    return {"status": "ok"}


@app.post("/query-kb")
async def query_kb(request: Request):
    """
    Called by Ananya's search_knowledge_base tool during voice calls.
    Searches Pinecone and returns relevant KB chunks.
    """
    data = await request.json()

    tool_call_id = data.get("message", {}).get("toolCallList", [{}])[0].get("id", "")
    arguments = data.get("message", {}).get("toolCallList", [{}])[0].get("function", {}).get("arguments", {})
    query = arguments.get("query", "")

    if not query:
        query = data.get("query", "")

    print(f"KB query: {query}")

    tenant = get_tenant()

    from kb.embeddings import search_chunks
    chunks = search_chunks(query=query, namespace=tenant["kb_namespace"], top_k=3)
    context = "\n".join([c["text"] for c in chunks])

    print(f"KB result preview: {context[:200]}")

    if tool_call_id:
        return {
            "results": [
                {
                    "toolCallId": tool_call_id,
                    "result": context
                }
            ]
        }

    return {"result": context}

import os
from pinecone import Pinecone
from dotenv import load_dotenv

load_dotenv()
pc=Pinecone(api_key=os.getenv("PINECONE_API_KEY"))
index = pc.Index(os.getenv("PINECONE_INDEX_NAME"))

def store_chunks(chunks:list[dict],namespace:str='troy_shop'):
    records=[]
    for chunk in chunks:
        records.append(
            {
                "id":chunk["id"],
                "text":chunk["text"],
                **chunk["metadata"]
            }
        )
    index.upsert_records(namespace=namespace, records=records)
    print(f"Stored {len(records)} chunks in namespace: {namespace}")

def search_chunks(query: str, namespace: str = "troy", top_k: int = 3):
    """
    query = customer's question
    top_k = how many matching chunks to return
    Returns list of matching text chunks
    """

    # search_records sends query text to Pinecone
    # Pinecone embeds query using same model
    # Returns top_k closest records
    results = index.search_records(
        namespace=namespace,
        query={"inputs": {"text": query}, "top_k": top_k},
        fields=["text", "page_title"]  # which metadata fields to return
    )

    # extract just the text from results
    chunks = []
    for match in results["result"]["hits"]:
        chunks.append({
            "text": match["fields"].get("text", ""),
            "page_title": match["fields"].get("page_title", "")
        })

    return chunks

def get_nango_headers() -> dict:
    """
    Returns headers needed for Nango proxy calls.
    Nango uses these to inject the correct OAuth token automatically.
    No need to fetch token manually — Nango handles it.
    """
    return {

        "Connection-Id": os.getenv("NOTION_CONNECTION_ID"),
        "Provider-Config-Key": os.getenv("NOTION_INTEGRATION_ID"),
        "Content-Type": "application/json"
    }

def get_nango_proxy_url(endpoint: str) -> str:
    """
    Builds Nango proxy URL for any Notion API endpoint.
    e.g. endpoint = "/v1/pages/abc123"
    returns "https://api.nango.dev/proxy/v1/pages/abc123"
    """
    return f"{os.getenv('NANGO_BASE_URL')}/proxy{endpoint}"

import os
import asyncio
from notion_client import Client
from langchain_classic.text_splitter import RecursiveCharacterTextSplitter
from kb.embeddings import store_chunks
from dotenv import load_dotenv

load_dotenv()

# use token directly from .env
notion = Client(auth=os.getenv("NOTION_TOKEN"))

splitter = RecursiveCharacterTextSplitter(
    chunk_size=500,
    chunk_overlap=50
)

def extract_text_from_blocks(blocks: list) -> str:
    text = ""
    for block in blocks:
        block_type = block["type"]
        supported_types = [
            "paragraph",
            "heading_1", "heading_2", "heading_3",
            "bulleted_list_item", "numbered_list_item"
        ]
        if block_type in supported_types:
            rich_text = block[block_type]["rich_text"]
            for text_obj in rich_text:
                text += text_obj["plain_text"] + " "
        text += "\n"
    return text.strip()


def fetch_page_text(page_id: str) -> tuple[str, str]:
    page = notion.pages.retrieve(page_id=page_id)

    title = ""
    props = page["properties"]
    for prop in props.values():
        if prop["type"] == "title":
            for text_obj in prop["title"]:
                title += text_obj["plain_text"]
            break

    blocks_response = notion.blocks.children.list(block_id=page_id)
    blocks = blocks_response["results"]
    text = extract_text_from_blocks(blocks)
    return title, text


def sync_page_to_pinecone(page_id: str, namespace: str = "superowl-proto"):
    print(f"Fetching page {page_id} from Notion...")
    title, text = fetch_page_text(page_id)

    if not text.strip():
        print(f"Page {page_id} is empty, skipping.")
        return

    print(f"Page title: {title}")
    print(f"Text length: {len(text)} chars")

    text_chunks = splitter.split_text(text)
    print(f"Split into {len(text_chunks)} chunks")

    chunks = []
    for i, chunk_text in enumerate(text_chunks):

            "metadata": {
                "page_id": page_id,
                "page_title": title,
                "chunk_index": i
            }
        })

    store_chunks(chunks, namespace=namespace)
    print(f"Synced '{title}' → {len(chunks)} chunks stored")

===== vapi_migrate.py =====
"""
vapi_migrate.py — SuperOwl VAPI Account Migration Tool
=======================================================
EXPORT: pulls all assistants, tools, phone numbers from current account
IMPORT: recreates everything on a new account, remaps IDs, updates .env

Usage:
    Export:  uv run python vapi_migrate.py export
    Import:  uv run python vapi_migrate.py import --key sk-... --ngrok https://your-ngrok-url.ngrok-free.app
    Both:    uv run python vapi_migrate.py export && uv run python vapi_migrate.py import --key sk-... --ngrok https://...

What you must do manually on new account:
  1. Sign up at vapi.ai → copy API key
  2. Add Twilio credentials in VAPI dashboard (Phone Numbers → Add Provider)
  3. Buy or import a Twilio number
  4. Run: python vapi_migrate.py import --key <new_api_key> --ngrok <new_ngrok_url>

The script will handle everything else.
"""

import httpx
import json
import os
import sys
import argparse
from pathlib import Path
from dotenv import load_dotenv, set_key

load_dotenv()

EXPORT_DIR = Path("vapi_export")
ENV_FILE = Path(".env")
BASE_URL = "https://api.vapi.ai"

# Fields VAPI rejects if sent on creation (read-only)
STRIP_FIELDS = {"id", "orgId", "createdAt", "updatedAt","isServerUrlSecretSet"}


# ─── HELPERS ─────────────────────────────────────────────────────────────────

def get_headers(api_key: str) -> dict:
    return {
        "Authorization": f"Bearer {api_key}",
        "Content-Type": "application/json"
    }


def vapi_get(path: str, api_key: str) -> list | dict:
    r = httpx.get(f"{BASE_URL}/{path}", headers=get_headers(api_key), timeout=30)
    r.raise_for_status()
    return r.json()


def vapi_post(path: str, payload: dict, api_key: str) -> dict:
    r = httpx.post(f"{BASE_URL}/{path}", headers=get_headers(api_key), json=payload, timeout=30)
    if not r.is_success:
        print(f"  ✗ POST /{path} failed ({r.status_code}):")
        print(f"     {r.text}")
        return {}
    return r.json()


def strip(obj: dict) -> dict:
    """Remove read-only fields from a resource before re-creating."""
    return {k: v for k, v in obj.items() if k not in STRIP_FIELDS}


def save_json(name: str, data):
    EXPORT_DIR.mkdir(exist_ok=True)
    path = EXPORT_DIR / f"{name}.json"
    with open(path, "w") as f:
        json.dump(data, f, indent=2)
    print(f"  Saved → {path}")


def load_json(name: str):
    path = EXPORT_DIR / f"{name}.json"
    if not path.exists():
        print(f"  ✗ {path} not found — run export first")
        sys.exit(1)
    with open(path) as f:
        return json.load(f)


def tool_identity_keys(tool: dict) -> list[str]:
    """
    Return stable identity keys for matching tools across accounts.
    Returns keys in priority order: most specific first.
    """
    keys: list[str] = []

    # Most specific: function name (only function tools have this)
    fn_name = tool.get("function", {}).get("name")
    if fn_name:
        keys.append(f"fn:{fn_name}")

    # Webhook URLs are highly specific
    server_url = tool.get("server", {}).get("url")
    if server_url:
        keys.append(f"server_url:{server_url}")

    # API request tools
    api_url = tool.get("url")
    if api_url:
        keys.append(f"api_url:{api_url}")

    # Top-level name (fallback)
    top_name = tool.get("name")
    if top_name:
        keys.append(f"name:{top_name}")

    # Tool type (least specific — only as last resort)
    tool_type = tool.get("type")
    if tool_type:
        keys.append(f"type:{tool_type}")

    return keys


# ─── EXPORT ──────────────────────────────────────────────────────────────────

def export(api_key: str):
    print("\n🔽 EXPORTING from current VAPI account...\n")

    # Tools
    print("📦 Tools...")
    tools = vapi_get("tool", api_key)
    save_json("tools", tools)
    print(f"  {len(tools)} tool(s) exported")

    # Assistants
    print("\n🤖 Assistants...")
    assistants = vapi_get("assistant", api_key)
    save_json("assistants", assistants)
    print(f"  {len(assistants)} assistant(s) exported")

    # Phone numbers (metadata only — number itself tied to Twilio)
    print("\n📞 Phone numbers...")
    phones = vapi_get("phone-number", api_key)
    save_json("phone_numbers", phones)
    print(f"  {len(phones)} phone number(s) exported")

    # Save ID reference map for humans to read
    id_ref = {
        "tools": {t.get("function", {}).get("name", t.get("name", t["id"])): t["id"] for t in tools},
        "assistants": {a["name"]: a["id"] for a in assistants},
        "phone_numbers": {p.get("number", p["id"]): p["id"] for p in phones},
    }
    save_json("id_reference", id_ref)

    print("\n✅ Export complete. Files saved to ./vapi_export/")
    print("   id_reference.json shows all resource names → IDs for reference\n")


# ─── IMPORT ──────────────────────────────────────────────────────────────────

def import_to_new_account(new_api_key: str, new_ngrok_url: str, skip_webhooks: bool = False):
    print("\n🔼 IMPORTING to new VAPI account...\n")
    if skip_webhooks:
        print("⚠ Webhook validation skipped — tools will import without server URLs.\n")

    # Normalise ngrok URL
    new_ngrok_url = new_ngrok_url.rstrip("/")

    id_map = {}  # old_id → new_id
    tools_missing_webhooks = []  # Track tools that had webhooks stripped

    # ── 1. Import tools ──────────────────────────────────────────────────────
    print("📦 Importing tools...")
    tools = load_json("tools")

    # Get existing tools on new account
    try:
        existing_tools = vapi_get("tool", new_api_key)
        existing_tool_index = {}
        for t in existing_tools:
            # Only index by MOST SPECIFIC key to prevent collisions
            keys = tool_identity_keys(t)
            if keys:
                existing_tool_index.setdefault(keys[0], t["id"])
        print(f"  Found {len(existing_tools)} tool(s) already on new account")
    except Exception as e:
        print(f"  ⚠ Could not fetch existing tools: {e}")
        existing_tool_index = {}

    for tool in tools:
        old_id = tool["id"]
        tool_name = tool.get("function", {}).get("name", tool.get("name", old_id))

        # Skip if tool already exists on new account (idempotent import)
        # Match by MOST SPECIFIC key only to avoid collisions
        keys = tool_identity_keys(tool)
        matched_key = None
        existing_id = None
        if keys and keys[0] in existing_tool_index:
            matched_key = keys[0]
            existing_id = existing_tool_index[matched_key]

        if existing_id:
            id_map[old_id] = existing_id
            print(
                f"  ↷ Tool '{tool_name}' already exists by {matched_key}: "
                f"{existing_id[:8]}... (skipped)"
            )
            continue

        payload = strip(tool)

        # Update webhook URLs to new ngrok
        if "server" in payload and "url" in payload["server"]:
            if skip_webhooks:
                # Strip server config entirely to allow tool creation without reachable webhooks
                del payload["server"]
                tools_missing_webhooks.append(tool_name)
                print(f"  Webhook stripped")
            else:
                old_url = payload["server"]["url"]
                # Replace just the base domain, keep the path
                path_part = "/" + "/".join(old_url.split("/")[3:])
                payload["server"]["url"] = new_ngrok_url + path_part
                print(f"  Webhook updated: {path_part}")

        result = vapi_post("tool", payload, new_api_key)
        new_id = result.get("id")

        if new_id:
            id_map[old_id] = new_id
            # Update local index to avoid duplicate creation in same import run.
            # Only index by the MOST SPECIFIC key to prevent collisions
            keys = tool_identity_keys(payload)
            if keys:
                existing_tool_index.setdefault(keys[0], new_id)
            print(f"  ✓ Tool '{tool_name}': {old_id[:8]}... → {new_id[:8]}...")
        else:
            print(f"  ✗ Tool '{tool_name}' failed to import")

    # ── 2. Import assistants ─────────────────────────────────────────────────
    print("\n🤖 Importing assistants...")
    assistants = load_json("assistants")

    # Get existing assistants on new account
    try:
        existing_assistants = vapi_get("assistant", new_api_key)
        existing_asst_names = {a["name"]: a["id"] for a in existing_assistants}
        print(f"  Found {len(existing_asst_names)} assistant(s) already on new account")
    except Exception as e:
        print(f"  ⚠ Could not fetch existing assistants: {e}")
        existing_asst_names = {}

    for asst in assistants:
        old_id = asst["id"]

        # Skip if assistant already exists on new account
        if asst["name"] in existing_asst_names:
            existing_id = existing_asst_names[asst["name"]]
            id_map[old_id] = existing_id
            print(f"  ↷ Assistant '{asst['name']}' already exists: {existing_id[:8]}... (skipped)")
            continue

        payload = strip(asst)

        # Remap toolIds in model
        if "model" in payload and "toolIds" in payload["model"]:
            old_ids = payload["model"]["toolIds"]
            new_ids = [id_map.get(tid, tid) for tid in old_ids]

            # Validate all toolIds were remapped
            for old_tid, new_tid in zip(old_ids, new_ids):
                if old_tid not in id_map:
                    print(f"  ✗ Assistant '{asst['name']}' references missing tool {old_tid[:8]}...")
                    continue

            payload["model"]["toolIds"] = new_ids
            print(f"  Remapped {len(old_ids)} toolId(s) for '{asst['name']}'")

        # Update server webhook URL if present
        if "server" in payload and "url" in payload["server"]:
            old_url = payload["server"]["url"]
            path_part = "/" + "/".join(old_url.split("/")[3:])
            payload["server"]["url"] = new_ngrok_url + path_part

        result = vapi_post("assistant", payload, new_api_key)
        new_id = result.get("id")

        if new_id:
            id_map[old_id] = new_id
            print(f"  ✓ Assistant '{asst['name']}': {old_id[:8]}... → {new_id[:8]}...")
        else:
            print(f"  ✗ Assistant '{asst['name']}' failed to import")

    # ── 3. Phone number — list what's on new account ─────────────────────────
    print("\n📞 Phone numbers...")
    print("  ⚠ Phone numbers cannot be auto-created (tied to Twilio credentials).")
    print("  Checking what's already on the new account...")

    try:
        new_phones = vapi_get("phone-number", new_api_key)
        if new_phones:
            for p in new_phones:
                print(f"  Found: {p.get('number')} → ID: {p['id']}")
            new_phone_id = new_phones[0]["id"]
            new_phone_number = new_phones[0].get("number", "")
        else:
            print("  No phone numbers found on new account.")
            print("  → Add Twilio credentials + import number in VAPI dashboard first.")
            new_phone_id = "REPLACE_ME"
            new_phone_number = "REPLACE_ME"
    except Exception as e:
        print(f"  Could not fetch phone numbers: {e}")
        new_phone_id = "REPLACE_ME"
        new_phone_number = "REPLACE_ME"

    # ── 4. Build new ID reference ─────────────────────────────────────────────
    new_id_ref = {}
    old_ref = load_json("id_reference")

    for category, name_to_old_id in old_ref.items():
        new_id_ref[category] = {}
        for name, old_id in name_to_old_id.items():
            new_id_ref[category][name] = id_map.get(old_id, "NOT_MAPPED")

    if new_phone_id != "REPLACE_ME":
        new_id_ref["phone_numbers"] = {new_phone_number: new_phone_id}

    save_json("id_reference_new_account", new_id_ref)

    # ── 5. Update .env ────────────────────────────────────────────────────────
    print("\n📝 Updating .env...")

    if not ENV_FILE.exists():
        print("  .env not found — creating it")
        ENV_FILE.touch()

    set_key(str(ENV_FILE), "VAPI_API_KEY", new_api_key)
    print("  ✓ VAPI_API_KEY updated")

    # Find assistant IDs by name
    old_assistants = load_json("assistants")
    for asst in old_assistants:
        old_id = asst["id"]
        new_id = id_map.get(old_id)
        if not new_id:
            continue
        name_lower = asst["name"].lower()
        if "ananya" in name_lower or name_lower == "ananya":
            set_key(str(ENV_FILE), "VAPI_ASSISTANT_ID", new_id)
            print(f"  ✓ VAPI_ASSISTANT_ID → {new_id}")
        elif "owner" in name_lower or "pa" in name_lower:
            set_key(str(ENV_FILE), "VAPI_OWNER_ASSISTANT_ID", new_id)
            print(f"  ✓ VAPI_OWNER_ASSISTANT_ID → {new_id}")

    if new_phone_id != "REPLACE_ME":
        set_key(str(ENV_FILE), "VAPI_PHONE_NUMBER_ID", new_phone_id)
        print(f"  ✓ VAPI_PHONE_NUMBER_ID → {new_phone_id}")
    else:
        print("  ⚠ VAPI_PHONE_NUMBER_ID not updated — set manually after adding Twilio number")

    # ── 6. Print summary ──────────────────────────────────────────────────────
    print("\n" + "="*55)
    print("✅ IMPORT COMPLETE")
    print("="*55)
    print(f"\nID map ({len(id_map)} resources migrated):")
    for old, new in id_map.items():
        print(f"  {old[:12]}... → {new[:12]}...")

    print("\nNew .env values set automatically.")

    if skip_webhooks and tools_missing_webhooks:
        print(f"\n⚠ {len(tools_missing_webhooks)} tool(s) imported WITHOUT webhooks:")
        for name in tools_missing_webhooks:
            print(f"  - {name}")
        print("\n  To add webhooks back:")
        print("  1. Start your backend with updated ngrok URL")
        print("  2. Edit each tool in VAPI dashboard → add server webhook URL")

    if new_phone_id == "REPLACE_ME":
        print("\n⚠ Still needed manually:")
        print("  1. Add Twilio credentials in VAPI dashboard")
        print("  2. Import/buy Twilio number")
        print("  3. Set VAPI_PHONE_NUMBER_ID in .env")
    print("\nRestart your backend and update ngrok URL — you're good to go.\n")


# ─── CLI ─────────────────────────────────────────────────────────────────────

def main():
    parser = argparse.ArgumentParser(
        description="SuperOwl VAPI Migration Tool",
        formatter_class=argparse.RawDescriptionHelpFormatter,
        epilog="""
Examples:
  Export current account:
    uv run python vapi_migrate.py export

  Import to new account:
    uv run python vapi_migrate.py import --key sk-abc123 --ngrok https://xyz.ngrok-free.app

  Full migration (export then import):
    uv run python vapi_migrate.py export
    uv run python vapi_migrate.py import --key sk-abc123 --ngrok https://xyz.ngrok-free.app
        """
    )
    subparsers = parser.add_subparsers(dest="command")

    # Export command
    export_parser = subparsers.add_parser("export", help="Export from current account")
    export_parser.add_argument(
        "--key",
        default=os.getenv("VAPI_API_KEY"),
        help="VAPI API key (defaults to VAPI_API_KEY in .env)"
    )

    # Import command
    import_parser = subparsers.add_parser("import", help="Import to new account")
    import_parser.add_argument("--key", required=True, help="New account VAPI API key")
    import_parser.add_argument("--ngrok", required=True, help="New ngrok base URL (e.g. https://xyz.ngrok-free.app)")
    import_parser.add_argument(
        "--skip-webhooks",
        action="store_true",
        help="Strip webhook/server config from tools to allow instant import (add webhooks manually later)"
    )

    args = parser.parse_args()

    if args.command == "export":
        if not args.key:
            print("✗ No API key found. Pass --key or set VAPI_API_KEY in .env")
            sys.exit(1)
        export(args.key)

    elif args.command == "import":
        import_to_new_account(args.key, args.ngrok, skip_webhooks=args.skip_webhooks)

    else:
        parser.print_help()


if __name__ == "__main__":
    main()

import json
import os

DB_FILE = "businesses.json"

def get_business(phone: str):
    if not os.path.exists(DB_FILE):
        return None

    try:
        with open(DB_FILE, "r") as f:
            content = f.read().strip()
            if not content:
                return None
            data = json.loads(content)
            return data.get(phone)
    except json.JSONDecodeError:
        print(f"❌ Error reading {DB_FILE}, file may be corrupted")
        return None

def save_business(phone: str, connection_id: str, display_name: str, slack_channel_id: str):
    data = {}
    if os.path.exists(DB_FILE):
        try:
            with open(DB_FILE, "r") as f:
                content = f.read().strip()
                if content:
                    data = json.loads(content)
        except json.JSONDecodeError:
            print(f"⚠️ Corrupted {DB_FILE}, creating new one")
            data = {}

    data[phone] = {
        "display_name": display_name,
        "nango_connection_id": connection_id,
        "nango_integration_id": "slack",
        "slack_channel_id": slack_channel_id,
        "system_prompt": f"You are a helpful assistant for {display_name}."
    }

    with open(DB_FILE, "w") as f:
        json.dump(data, f, indent=4)

from fastapi import APIRouter, Request
from app.core.state import call_sessions
import asyncio, httpx

router = APIRouter()

@router.post("/slack/events")
async def slack_events(request: Request):
    body = await request.json()

    # 1. Handle Slack URL verification challenge
    if body.get("type") == "url_verification":
        challenge = body.get("challenge")
        print("Slack URL Verified!!")
        return {"challenge": challenge}

    event = body.get("event", {})

    # 2. Handle message events
    event_type = event.get("type")
    text = event.get("text", "")
    thread_ts = event.get("thread_ts")
    bot_id = event.get("bot_id")

    # RULE: Must be a message, must be in a thread, and MUST NOT be from our own bot
    if event_type == "message" and thread_ts and not bot_id:
        print(f"\n🗣️ OWNER MESSAGE RECEIVED: {text}")

        # 3. Find which call this thread_ts belongs to
        target_call_id = None
        target_control_url = None

        for call_id, session_data in call_sessions.items():
            if session_data.get("slack_thread_ts") == thread_ts:
                target_call_id = call_id
                target_control_url = session_data.get("control_url")
                break

        if target_call_id and target_control_url:
            print(f"✅ Matched to Call ID: {target_call_id}")
            print(f"🔗 Ready to inject via Control URL: {target_control_url}")

            clean_text = text.lower().strip()

            # --- TAKEOVER ---
            if clean_text == "takeover" or clean_text == "!takeover":
                fallback_number = call_sessions[target_call_id]["config"].fallback_number
                payload = {
                    "type": "transfer",
                    "destination": {
                        "type": "sip",
                        "sipUri": f"sip:{fallback_number}@40942180.sip.vobiz.ai"
                    }
                }
                print("🔀 TAKEOVER payload built")

            # --- END CALL ---
            elif clean_text in ["end", "reject", "stop", "hangup"]:
                payload = {
                    "type": "end-call",
                    "message": {
                        "role": "system",
                        "content": (
                            "The owner has asked to end this call. "
                            "Say: 'I'm sorry, the owner is not available right now. Please call back later. Goodbye!' "
                            "Then call end_call_tool immediately."
                        )
                    }
                }
                print("🗣️ WHISPER payload built")

            # --- WHISPER (default) ---
            else:
                payload = {
                    "type": "add-message",
                    "message": {
                        "role": "system",
                        "content": f"The business owner just whispered: '{text}'. Naturally incorporate this into your next response to the customer."
                    }
                }
                print("🗣️ WHISPER payload built")

            # Fire the command to VAPI (works for all three cases)
            async def send_command():
                async with httpx.AsyncClient() as client:
                    try:
                        resp = await client.post(target_control_url, json=payload)
                        print(f"🎯 Command injected! Vapi replied: {resp.status_code}")
                    except Exception as e:
                        print(f"❌ Failed to inject command: {e}")

            asyncio.create_task(send_command())

        else:
            print("⚠️ Could not find an active call for this Slack thread.")

    # Always return 200 OK so Slack doesn't retry sending the message
    return {"status": "ok"}

import httpx
from typing import Any, Optional
from app.core.config import NANGO_SECRET_KEY, NANGO_BASE_URL  # Changed

async def post_to_slack_via_nango(config: Any, text: str, thread_ts: Optional[str] = None) -> Optional[str]:
    """Sends a message to Slack using Nango's Proxy endpoint."""
    url = f"{NANGO_BASE_URL}/proxy/chat.postMessage"
    headers = {
        "Authorization": f"Bearer {NANGO_SECRET_KEY}",
        "Connection-Id": config.nango_connection_id,
        "Provider-Config-Key": config.nango_integration_id,
        "Content-Type": "application/json",
    }
    payload = {"channel": config.slack_channel_id, "text": text}
    if thread_ts:
        payload["thread_ts"] = thread_ts

    async with httpx.AsyncClient() as client:
        try:
            resp = await client.post(url, headers=headers, json=payload)
            print(f"Nango proxy status: {resp.status_code}")
            print(f"Nango proxy response: {resp.text[:300]}")
            data = resp.json()
            if data.get("ok"):
                return data.get("ts")
            print(f"❌ Nango/Slack Error: {data}")
        except Exception as e:
            print(f"❌ Nango Proxy request failed: {e}")
    return None

async def background_slack_init(config: Any, text: str, call_id: str):

async def background_post_summary(session: dict, transcript_text: str):
    """Heavy lifting: LLM summarization and Slack posting in the background."""
    try:
        config = session.get("config")

        # Run the synchronous Groq call in a thread to prevent blocking the event loop
        def fetch_summary():
            return groq_client.chat.completions.create(
                model="llama-3.1-8b-instant",
                messages=[
                    {"role": "system", "content": "Summarize this call transcript professionally in 3-4 bullet points."},
                    {"role": "user", "content": f"Transcript:\n{transcript_text}"}
                ]
            )

        chat_completion = await asyncio.to_thread(fetch_summary)
        summary_text = chat_completion.choices[0].message.content

        await post_to_slack_via_nango(
            config=config,
            text=f"📋 *Final Call Summary:*\n{summary_text}",

        )

async def _handle_assistant_request(ctx: dict[str, Any]) -> Optional[dict[str, Any]]:
    call = ctx["call"]
    call_id = ctx["call_id"]
    customer_number = ctx["customer_number"]

    # GUARD: Prevent double initialization if VAPI retries the request
    if call_id in call_sessions and call_sessions[call_id].get("slack_thread_ts"):
        return None

    # Extract Owner Number (ANI) from SIP Diversion
    sip_headers = call.get("phoneCallProviderDetails", {}).get("sip", {}).get("headers", {})
    diversion = sip_headers.get("Diversion", "") or call.get("sipHeaders", {}).get("Diversion", "")

    match = re.search(r"sip:(\+?\d+)", diversion)
    owner_number = match.group(1) if match else "9611896916" # Fallback for testing

    # Lookup Business via Registry
    config, matched = get_config_by_ani(owner_number)
    print(f"📞 Call for: {config.display_name} (Connection: {config.nango_connection_id})")

    # Initialize session state
    call_sessions[call_id] = {
        "customer_number": customer_number,
        "slack_thread_ts": None,
        "config": config,
        "transcript": []
    }

    # Trigger Slack Alert in Background
    if config.nango_connection_id and config.slack_channel_id:
        msg = f"📞 *Incoming Call*\n*Business:* {config.display_name}\n*Customer:* {customer_number}"
        asyncio.create_task(background_slack_init(config, msg, call_id))

    # Return Assistant Config to Vapi
    if VAPI_BASE_ASSISTANT_ID:
        return {
            "assistantId": VAPI_BASE_ASSISTANT_ID,
            "assistantOverrides": {
                "variableValues": {
                    **config.system_prompt_var,
                    "callerNumber": customer_number
                },
                "firstMessage": config.first_message,
            }
        }

    return {
        "assistant": {
            "name": config.display_name,
            "firstMessage": config.first_message,
            "model": {
                "provider": "openai",
                "model": "gpt-4o-mini",
                "messages": [{"role": "system", "content": f"Assistant for {config.display_name}"}]
            }
        }
    }

def _handle_tool_calls(ctx: dict[str, Any]) -> Optional[dict[str, Any]]:
    message = ctx["message"]
    tool_calls = message.get("toolCallList", [])
    if not tool_calls:
        tool_calls = [
            entry.get("toolCall", {}) for entry in message.get("toolWithToolCallList", []) if entry.get("toolCall")
        ]

    results = []
    for tool_call in tool_calls:
        tool_name = tool_call.get("name", "")
        results.append({
            "name": tool_name,
            "toolCallId": tool_call.get("id", ""),
            "result": json.dumps({"status": "ok", "message": f"Tool {tool_name} executed"}),
        })
    return {"results": results}

async def _handle_end_of_call_report(ctx: dict[str, Any]) -> Optional[dict[str, Any]]:
    call_id = ctx["call_id"]

    # pop to clean up memory immediately
    session = call_sessions.pop(call_id, {})

    transcript = session.get("transcript", [])
    config = session.get("config")

    if not transcript or not config or not config.slack_channel_id:
        print(f"Call {call_id} ended — no transcript or Slack config")
        return None

    # Format transcript for LLM
    transcript_text = "\n".join([f"{entry['role'].capitalize()}: {entry['text']}" for entry in transcript])

    # Run the summary in the background so we can return 200 to VAPI instantly
    asyncio.create_task(background_post_summary(session, transcript_text))

    return {"status": "processing_summary"}

async def _handle_transcript(ctx: dict[str, Any]) -> Optional[dict[str, Any]]:
    message = ctx["message"]
    call_id = ctx["call_id"]
    role = message.get("role", "assistant")

    # VAPI sometimes sends transcript text in different fields based on the event
    text = message.get("transcript") or message.get("text") or ""

    # Send only final transcript segments to Slack thread AND save to memory
    if message.get("transcriptType") == "final" and call_id in call_sessions:
        session = call_sessions[call_id]

        if text:
            # FIX: This is now safely inside the "final" check!
            _append_transcript(call_id, role, text)

            if session.get("slack_thread_ts"):
                icon = "🤖" if role == "assistant" else "👤"
                asyncio.create_task(post_to_slack_via_nango(
                    config=session["config"],
                    text=f"{icon} *{role.capitalize()}:* {text}",
                    thread_ts=session["slack_thread_ts"]
                ))

    return None
